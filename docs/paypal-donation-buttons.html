PayPal Donation Buttons

Church: Tabernacle Amour Pour Le Calvaire

PayPal Email: amorporelcalvario@gmail.com

Currency: DOP (Dominican Peso)

📋 How to Use These Buttons

  1. Scroll down to find the button style you want
  2. Copy the HTML code from the gray code block
  3. Paste it into your website HTML where you want the button to appear
  4. Save and publish your website
  5. Test the button to make sure it works
⚠️ Important: After adding buttons to your website, remember to manually record donations in your admin panel when they come through!

💡 Tips for Using These Buttons

  • Test First: After adding to your website, test the button to make sure it works
  • Mobile Friendly: All buttons work on mobile devices
  • Customize Colors: Change the "background" color in the custom buttons to match your website
  • Track Categories: Use different buttons for different funds (Building, Missions, etc.) so you know where donations are meant to go
  • Record Manually: When you receive a PayPal notification, record it in your admin panel

⚠️ Important Reminders

  • Do NOT change the email address in the code - it must be: amorporelcalvario@gmail.com
  • Do NOT change the currency - it's set to DOP
  • Always record donations in your admin panel when you receive them
  • Save Transaction IDs from PayPal notifications
  • Test buttons before announcing to your congregation
