## 📅 Date Format (IMPORTANT!)

**Always use: YYYY-MM-DD**

Examples:
- ✅ 2024-06-15
- ✅ 2025-12-25
- ❌ 06/15/2024 (Wrong!)
- ❌ 15-06-2024 (Wrong!)

## 🚀 Quick Start Steps

1. **Download** template from `/templates/` folder
2. **Open** in Excel or Google Sheets
3. **Keep** header row unchanged
4. **Delete** sample data rows
5. **Fill in** your data
6. **Save** as CSV format
7. **Import** to system (feature coming soon)

---

## ⚠️ Common Errors to Avoid

| Error | Solution |
|-------|----------|
| Wrong date format | Use YYYY-MM-DD only |
| Modified headers | Keep original column names |
| Missing required fields | Fill all required fields |
| Wrong file format | Save as CSV, not Excel |
| Extra columns | Use only provided columns |
| Commas in text | Wrap text in quotes: "Text, with comma" |

---

## 💾 How to Save as CSV

**Excel:**
1. File → Save As
2. Select "CSV (Comma delimited) (*.csv)"
3. Click Save

**Google Sheets:**
1. File → Download
2. Select "Comma Separated Values (.csv)"
3. File downloads automatically
